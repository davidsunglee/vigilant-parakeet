# Apex Predator Confrontation

A generative educational narrative engine that creates illustrated storybooks in the style of Jerry Pallotta's *"Who Would Win?"* series. Users pick two animals, and the app generates a complete 26+ page book вҖ” with AI-written content, AI-generated illustrations, and a simulated battle outcome.

## вңЁ Features

- **AI Story Generation** вҖ” Full-length educational narratives generated via Google Gemini, covering 12 comparative aspects (habitat, size, weapons, speed, intelligence, etc.)
- **AI Illustrations** вҖ” Every page gets a unique, children's book-style illustration generated by `gemini-2.5-flash-image`
- **Dynamic Cover Art** вҖ” AI-generated cover image featuring both animals in an epic standoff
- **Interactive Book Viewer** вҖ” Page-flip reading experience powered by `react-pageflip`
- **Spoiler-Free Dashboard** вҖ” Winner is hidden behind a "Reveal Winner" toggle to preserve suspense
- **Offline Persistence** вҖ” All generated stories are stored locally via IndexedDB (`localforage`)
- **Surprise Endings** вҖ” 1-in-7 chance of a non-standard outcome (external event, mutual retreat, etc.)

### Data Model

The core data structure is `IStoryManifest`:

```typescript
IStoryManifest
в”ңв”Җв”Җ metadata          // id, title, createdAt, hasBeenRead
в”ңв”Җв”Җ animalA            // IAnimalEntity (name, scientific name, stats, habitat)
в”ңв”Җв”Җ animalB            // IAnimalEntity
в”ңв”Җв”Җ coverImageUrl?    // Base64 data URI for the AI-generated cover
в”ңв”Җв”Җ pages[]           // IPageContent[] вҖ” 26 pages (12 aspect pairs + showdown + outcome)
в”ңв”Җв”Җ checklist         // ITraitChecklist вҖ” per-trait advantage comparison
в””в”Җв”Җ outcome           // IBattleOutcome вҖ” winner, reasoning, ending type
```

### Book Structure (26 pages)

| Pages | Content |
|-------|---------|

## рҹҡҖ Getting Started

### Prerequisites

- Node.js 18+
- A [Google AI Studio](https://aistudio.google.com/) API key

### Setup

```bash
# Clone the repository
git clone https://github.com/your-username/vigilant-parakeet.git
cd vigilant-parakeet/apex

# Install dependencies
npm install

# Add your API key
echo "VITE_GEMINI_API_KEY=your_api_key_here" > .env

# Start development server
npm run dev
```

The app will be available at `http://localhost:5173`.

### Build

```bash
npm run build     # TypeScript check + Vite production build
npm run preview   # Preview the production build locally
```
