Add support for user-selectable art style for generated images.

Potential scope:
- Decide where the user specifies preferred art style in the flow
- Define which style options are supported and how open-ended the input can be
- Update image-generation prompts to incorporate the selected style
- Ensure the chosen style applies consistently within a generated story
- Consider how the cover should use the selected style versus any special cover treatment
- Verify manually that style choice meaningfully affects generated results
