Refine animal image generation so LLM-generated animal images feel more consistent across the app for a given story, without aiming for exact visual uniformity.

Scope
- Apply to all pages that include an animal image generated by LLM calls.
- This is for future generations only.
- The cover is exempt from consistency requirements beyond remaining child-friendly; it may use a more dynamic style because it depicts a clash of animals.

Primary goal
- Improve consistency of the depicted animals across pages within a single generated story.
- The most important consistency target is overall art style.
- Also preserve, as much as reasonably possible with LLM generation:
  - species / breed
  - fur / body colors
  - markings / patterns
  - face shape

Non-goals
- This is not about image size, aspect ratio, cropping, layout, or other physical formatting.
- Do not add reference-image / image-to-image conditioning as part of this task.
- Do not try to guarantee exact sameness across pages.
- Do not apply the consistency anchor across separate story generations; a new story generation may produce different animal looks, as long as each story is internally more consistent.

Implementation direction
- Treat this primarily as a prompt-engineering task.
- Investigate where image prompts are currently generated and whether there are multiple prompt-building paths contributing to inconsistency.
- Implement a shared base prompt / template to encourage a child-friendly, more consistent visual style across generated animal images.
- Add a dedicated one-time LLM step per generated story, before image generation, to produce a canonical animal description for the story.
- Reuse that canonical per-story description in later image prompts so animal depiction is more internally consistent across pages.
- Best-effort only; avoid heavy verification loops.

Constraints
- Extra-call budget is acceptable up to a low fixed number; up to 5 additional calls per story is acceptable.
- Avoid approaches that roughly double total generation/verification calls.

Verification
- Manual testing by the user is sufficient for this task.
- Success means images across pages within a story look more stylistically aligned and the animals retain more similar physical characteristics overall.
